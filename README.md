# üè† Parametric Shed Project Board

A simple Kanban board to track tasks, ideas, and progress for the Parametric Shed configurator project.

## Features

- **Drag & drop** cards between columns
- **4 status columns**: Ideas, Backlog, In Progress, Done
- **Priority levels**: High, Medium, Low (with color coding)
- **Persistent storage** using localStorage
- **Mobile friendly** responsive design
- **No dependencies** - pure HTML/CSS/JS

## Usage

1. Click **+ Add Card** to create a new task
2. **Drag cards** between columns to update status
3. **Click a card** to edit or delete it
4. Data is saved automatically to your browser's localStorage

## Live Demo

[View the board](https://andrewsgparsons-source.github.io/shed-project-board/)

## Project Context

This board tracks development of the [Parametric Shed Configurator](https://github.com/andrewsgparsons-source/Parametric-shed2) - a 3D garden building configurator built with Babylon.js.

## Local Development

Just open `index.html` in a browser - no build tools needed.
